# /status - 報告專案進度

當使用者說「報告專案進度」、「目前進度」或使用 /status 指令時，請執行以下動作：

## 自動執行步驟

1. **讀取專案文件**
   - 讀取 `PROJECT_STATUS.md`
   - 讀取 `PROJECT_LOG.md`（最近 5 條記錄）
   - 讀取 `PRD.md`（如需要確認功能範圍）

2. **產生進度報告**
   - 整體進度百分比與視覺化
   - 各階段進度
   - 已完成項目摘要
   - 進行中項目
   - 待開始的下一步
   - 已知問題（如有）
   - 最近的日誌摘要

3. **提供建議**
   - 根據目前進度，建議下一步優先處理什麼
   - 提醒可能的風險或阻礙
